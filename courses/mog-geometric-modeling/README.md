# MOG — Geometric Modeling

**Course:** Modelagem Geométrica  
**Program:** Master's in Applied Computing (PPGCAP) – UDESC  
**Credits:** 4

## Purpose

This folder is the working area for the Geometric Modeling course. It is organized to support Python 3 exercises for curves, graphs, surfaces, and other modeling experiments while keeping each activity isolated and easy to review.

## Local Environment

Create a virtual environment inside this folder and use it for all course work:

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## Dependencies

The current stack includes:

- `numpy` for numeric computation
- `matplotlib` for plots and curve visualization

Initial workspace scaffold created. The next steps are to add more activities and expand the course materials throughout the semester.
