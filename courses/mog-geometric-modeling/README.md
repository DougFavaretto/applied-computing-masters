# MOG — Geometric Modeling

**Course:** Modelagem Geométrica  
**Program:** Master's in Applied Computing (PPGCAP) – UDESC  
**Credits:** 4

## Description
Introduction to geometric modeling concepts and techniques. Topics include modeling and creation methods, Euler operators, Boolean operations, curves in the plane and in space, generation and representation of surfaces and solids, construction of complex models, neutral interfaces for file exchange, and advanced topics in geometric modeling.
