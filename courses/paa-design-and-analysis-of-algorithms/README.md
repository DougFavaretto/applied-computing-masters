# PAA — Design and Analysis of Algorithms

**Course:** Projeto e Análise de Algoritmos  
**Program:** Master's in Applied Computing (PPGCAP) – UDESC  
**Credits:** 4

## Description
Study of data structures and algorithms including lists, stacks, queues, trees and graphs. Concepts of asymptotic complexity such as O, Ω and Θ notation, induction, summations and recurrence relations. Algorithm design paradigms and an introduction to NP-complete problems.
