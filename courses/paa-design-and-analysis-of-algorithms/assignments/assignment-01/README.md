# Assignment 01

## Problem

Develop an algorithm that receives a sequence of `n` integers and then
receives a number `m`. The algorithm must output how many times the
value `m` appears in the sequence.

Constraints:
- n < 1,000,000

## Questions

- Is there any difference if `m` is between `0` and `10,000` or between
`0` and `1,000,000,000,000`?
- What changes if `m` is provided before the sequence?

## Note

Create an entry file with random values to test the algorithm.
